      How Scores Are Assembled

      The newsroom combines EfficientNet-B0 CNN probabilities (90% weight) with AutoEncoder reconstruction error (10% weight). Scores surpassing 0.15 are front-page "FAKE" alerts.

      Error Handling

      Images lacking discernible faces or broken videos surface gentle notices instead of crashes. All processing occurs locally — models from models/cnn_efficientnet_b0.pt and models/ae.pt are reused verbatim.

      One-Command Run

      Activate the Flask newsroom with python ui/app.py after installing requirements. Access via http://localhost:8000.
